# cook-for-you


## Run locally
```
hugo server -D

hugo server --disableFastRender

```
## Create new

> Create single new recipe file

```hugo new --kind recipe recipes/{category}/{my-new-recipe-name}/index.md```

> Create new recipes category

```hugo new --kind recipes recipes/{category}/_index.md```
